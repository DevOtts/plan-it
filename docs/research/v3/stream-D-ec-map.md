## TL;DR

- Root and `plugins/plan-it/` copies of every shared file are byte-identical but NOT symlinked (`diff` confirms) — every v3 change must be applied twice, and there's currently zero automated check that they stay in sync (mechanism #5 above fixes this).

- `gate-check.mjs`'s 4-subcommand `commands` map is trivially extensible — nearly every "script" row in the backlog (testconv, audit, liveness, frontmatter, scaffold, mirror-check) is a pure append, zero risk to the 4 existing checks, and can reuse `tests/run-contract.mjs`'s existing fixture-based harness for regression coverage.
- **FD-2 has a real structural mismatch**: the binding per-epic Test Contracts don't exist until Phase 9, which runs *after* both G2 and G3.
